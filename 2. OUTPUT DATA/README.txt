CFin.dat --> Final configuration
CIniA.dat -->Initial configuration
DT.dat -->Diffusion coefficient D(t)
EP.dat --> Mean potential energy for particles
GDR.dat --> Radial distribution function G(r)
Infectadas.dat --> Number of acumulated infected people each simulation step
MATRIZ_INFECCION.dat --> Infection matrix (rows are for particles and columns are for simulation step). 0 is not infected and 1 is infected.
WT.dat --> Mean square displacement W(t)
X.dat --> Matrix of X coordinate for every particle in each saved simulation step (rows are particles, columns are saved simulation steps)
X.dat --> Matrix of Y coordinate for every particle in each saved simulation step (rows are particles, columns are saved simulation steps)
